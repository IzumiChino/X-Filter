# X-Filter

纯前端、纯本地、自学习的 **X (Twitter) 客户端垃圾推文过滤器**。Chrome / Firefox 浏览器扩展。

所有学习样本与模型权重保存在浏览器本地 (`chrome.storage.local`)，不发起任何外部网络请求（云同步为手动触发）。

- 当前版本：**v1.0.2**
- 作者：好奇猫a & Izumi Chino
- 适配域名：`x.com` / `twitter.com`

---

## 安装

### Chrome / Edge

1. 下载 `x-filter-v1.0.2-chrome.zip` 并解压
2. 打开 `chrome://extensions`，开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择解压后的目录
4. 刷新 `x.com`，工具栏出现 X-Filter 图标、页面左下角出现蓝色 **XLS** 浮动按钮即代表安装成功

### Firefox

1. 下载 `x-filter-v1.0.2-firefox.xpi`
2. 打开 `about:debugging` →「此 Firefox」→「临时加载附加组件」
3. 选择 `.xpi` 文件
4. 刷新 `x.com`

## 工作原理：七级判定流水线

命中即停，按优先级从上到下：

| 顺序 | 通道 | 说明 |
| --- | --- | --- |
| 1 | **关注免屏蔽** | 鼠标悬浮用户卡片时，识别 "Following / 已关注" 按钮加入白名单 |
| 2 | **结构通道** | 纯 emoji / 纯数字 / emoji+数字 直接拦截 |
| 3 | **软诱导组合通道** | 识别情绪化、暧昧式短文本并结合账号与上下文评分 |
| 4 | **低信息短回复通道** | 识别模板化、重复化、低信息量短回复并结合账号与上下文评分 |
| 5 | **启发式通道** | 零训练即可工作，基于 handle 数字比、尾部数字、粉丝/关注比、bio 缺失等加权打分 |
| 6 | **ML 通道** | 33 维基础特征 + 256 维 n-gram hashing 的在线逻辑回归，样本不足时自动回退 |
| 7 | **Jaccard 回退** | n-gram + Jaccard 相似度对比黑/白样本库 |

### ML 特征（33 维 + hashing）

文本长度、emoji 比、数字比、内容风险分数、URL、字符多样性、重复度、handle 数字/熵/尾随数字、名字 emoji 比、是否回复、是否含媒体、Jaccard 黑白相似度、handle 声誉、粉丝/关注数、粉关比、bio/认证、联系方式/色情/诈骗锚点、中英混排、强风险锚点、软诱导分数、账号可疑度、软诱导组合、低信息分数、低信息组合。

## 使用

### 标记模式

每条推文旁注入 **"标注垃圾 / 标注正常"** 按钮，点击即更新样本库 + 更新 handle 声誉 + 触发本地重训。

ML 需要总样本 ≥ `mlMinSamples`（默认 8），且垃圾/正常各 ≥ 3 条才会激活。

### 屏蔽方式

- **折叠卡片**（默认）：替换为提示卡片，可展开或标注正常
- **直接隐藏**：`display:none`

### 控制面板

- **页面内面板**：左下角蓝色 **XLS** 浮动按钮 → 模态面板
- **工具栏弹窗**：点击扩展图标 → 设置弹窗

功能：模式开关、参数调节、实时统计、导入/导出/重训、云同步。

### 云同步

弹窗「数据同步」区域支持：

| 功能 | 说明 |
| --- | --- |
| 加载默认规则 | 从扩展内置 `rules.json` 加载预置规则 JSON |
| 从 URL 导入 | 输入任意托管的 JSON URL 合并数据 |
| GitHub Gist 双向同步 | 填入 Token + Gist ID，推送/拉取数据 |

## 数据存储

全部使用 `chrome.storage.local` 存在浏览器本地：

| Key | 内容 |
| --- | --- |
| `xls_samplesBad_v4` | 黑样本（带 n-gram 指纹） |
| `xls_samplesGood_v4` | 白样本 |
| `xls_trainData_v1` | ML 训练集（最多 500 条） |
| `xls_modelState_v1` | ML 模型权重 |
| `xls_followedHandles_v1` | 已关注白名单 |
| `xls_accountCache_v1` | 账号特征缓存（TTL 7 天） |
| `xls_handleRep_v1` | handle 声誉滚动均值（最近 20 次） |

### 导入 / 导出

导出 JSON 包含 `_meta`（统计摘要）、`config`（带默认值对比）、`data`（所有数据），2 空格缩进可直接阅读。导入自动去重合并。

## 调优建议

- 刚装好时 ML 未训练，主要靠**结构通道 + 软诱导/低信息/启发式通道**工作；标注 3 条垃圾和 3 条正常后 ML 激活
- 误伤多 → 标注正常样本 + 调高 `mlThresholdReply`；漏判多 → 标注垃圾样本 + 调低阈值
- 加载默认规则可快速获得基础拦截能力
